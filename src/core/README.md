# Core Module

Core Business Logic Layer - Application algorithms and data structures.

## Responsibilities

- Implement Binary Search Tree (BST)
- City sorting and ordering
- Data structure operations
- Core algorithms
- Business logic processing

## Binary Search Tree Operations

- **Insert**: Add cities in alphabetical order
- **Search**: Find cities by name
- **Remove**: Delete cities from the tree
- **Traversal**: In-order (sorted), pre-order, post-order
- **Height**: Calculate tree height
- **Count**: Count total nodes
- **Balance**: (Future) Balance the tree

## Public Header Files

Public interfaces are in the `include/` directory at project root.

## Design Principles

- Memory safety: Proper allocation and deallocation
- Error handling: Return NULL or error codes on failure
- Const correctness: Use const for read-only parameters
- Encapsulation: Hide internal implementation details
